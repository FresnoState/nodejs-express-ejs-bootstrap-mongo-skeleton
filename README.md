## File Structure
```
--- public (static pages)
 |
 +- routes (components)
 |
 +- uploads (make sure to chmod 777)
 |
 +- views (templates /ejs)
 |
 +- (mapping) /bootstrap -> node_modules/bootstrap/dist)
```

 - helloMaxExample is to test static content
